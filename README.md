# Simple Weather Application

This is a simple web-based Weather Application built using HTML, CSS, and JavaScript.  
The application allows the user to enter a city name and displays basic weather information such as temperature and weather condition by fetching data from a public weather API.

## Features

- User can enter a city name
- Displays current temperature in Celsius
- Displays current weather condition
- Uses real-time data from a public weather API
- Simple and beginner-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript
- WeatherAPI.com (Public Weather API)

## API Used

WeatherAPI  
https://www.weatherapi.com/

## How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Open `script.js` and make sure a valid API key is added
4. Open `index.html` in any modern web browser
5. Enter a city name and click **Get Weather**
